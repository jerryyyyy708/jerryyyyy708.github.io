簡介

編曲軟體遇到不支援的檔案類型時，時常會需要上網進行轉檔，這是非常麻煩的，
因此針對這個問題設計了VST Plugin，可以根據需求轉換音檔類型(比如 m4a -> mp3)，
讓使用者可以在不離開編曲軟體的情況下，直接透過此plugin進行轉檔， 進而節省轉檔所帶來的額外工作量。
GitHub Repository

目前已完成檔案轉換的功能，接下來須完成GUI的設計並將功能串接上去。

Build the Code

This part shows how to build the exe with the converting functionality.

Step 1.
Clone the GitHub repository

git clone https://github.com/jerryyyyy708/Audio_File_Converter.git
Step 2. Build the exe file using Makefile. (It might take a few minutes to complete the build.)
make
After that, you should see the converter.exe in the folder.
Usage 
./converter [input_file] [output_file]

實作細節

Converter.exe

  • 使用ffmpeg的轉換功能來進行檔案轉換。
  • 由於希望在功能embed於VST時，不需要依賴資料夾內的ffmpeg.exe，因此在build時使用exe2str.py將exe檔存入標頭檔ffmpeg_bytes.h。
  • 程式執行時會從header取得ffmpeg.exe的資訊，暫時建出ffmpeg並進行convert，並在執行完成後將暫存檔刪除。
  • Compile完成後，暫存的標頭檔會被移除，產出的converter.exe不須依賴ffmpeg或標頭檔。
