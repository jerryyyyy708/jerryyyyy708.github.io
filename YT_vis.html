YT收益分析與視覺化

Dataset

資料來自高中至大一經營的YouTube頻道真實數據。

資料視覺化

利用 D3.js 建立 Histogram、Scatter Plot 以及 Parrelel Coordinate Plot 幫助分析各項指標對收益的影響。
Demo Page

ML 數據分析

經過視覺化篩選出適合用於預測收益的feature後，利用Sklearn進行數據分析。

  • Decision Tree: 收益每1000元分為4個區間進行classification，準確率65%
  • RandomForest: 收益每1000元分為4個區間進行classification，準確率74%
  • KNN: 收益每1000元分為4個區間進行classification，準確率72%
  • Linear Regression: 預測實際收益數字，準確率73%
